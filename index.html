FORMULÁRIO

    2) Responda as perguntas abaixo:

  • Na tag <input> com atributo type="text", para que serve o atributo "placeholder"?
    • É usado para indicar ao navegador o texto que deverá aparecer escrito no campo do formulário, muitas vezes é para indicar ao usuário como deve ser preenchido aquele campo específico.
  • Na tag input com atributo type="radio" ou type="radio", para que serve o atributo "checked"?
    • O atributo checked quando é utilizado especifica que um elemento <input> deve ser pré-selecionado (verificado) quando a página for carregado. Além disso, pode ser usado no <input type= “checkbox”> e <input type= “radio”>
  • Na tag input para que serve o atributo "value"?
    • O atributo value especifica o valor de um elemento <input>, é usado de formas diferentes para tipos diferentes de entrada: para “button”, “reset” e“submit” define o texto no botão; para “text”, “password”, define o valor inicial (padrão) do campo de entrada; para “checkbox”, “radio” e “image” define o valor associado à entrada (que também é o valor enviado no submit).
  • Que atributo deve ser utilizado para agrupar vários botões de rádio em um formulário de modo que apenas um deles fique marcado de cada vez?
    • Com o atributo de “radio” pode fazer vários “radio button” e por meio do atribudo value que pode ser usado para escolher apenas uma opção, dentre as várias listadas.
  • Para que serve a tag label? Para que serve o atributo "for" nessa tag?
    • A tag <label> define um rótulo para vários elementos. E o atributo “for” nessa tag especifica o id do elemento ao qual o rótulo deve ser vinculado.
  • Na tag option para que serve o atributo "selected"?
    • O atributo "selected" serve para especificar uma opção que pode ser pré selecionada no carregamento da página.
  • Nas tags de formulário para que serve o atributo "required"?
    • Para informar que aquele é um campo obrigatório.
  • Na tag form para que serve o atributo "action"?
    • Action serve para dizer para onde a informação digitada no formulário será enviada.
  • Na tag form para que serve o atributo "method"?
    • Insere a informação digitada no campo na URL da página.
  • Para que o valor digitado de uma senha não apareça na barra de endereços do navegador, qual deve ser o valor do atributo "method" na tag form?
    • Deve ser atribuído o valor type="password"
  • Qual valor deve ser informado no atributo "method" na tag form em um formulário que será usado para fazer upload de arquivos?
